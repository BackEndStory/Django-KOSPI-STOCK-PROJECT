# Django_KOSPI_STOCK

### 주요 사용 모듈
1. django
2. matplotlib
3. FinanceDataReader
4. FileResponse
5. platform


### 포함 된 기능 
- 로그인,회원가입
- 아이디 찾기(해당 정보 입력 시 그 유저의 메일로 아이디 전송
- 비밀번호 찾기(해당 정보 입력 시 그 유저의 메일로 임시 비밀번호를 전송)
- 비밀번호 변경
- FinanceDataReader이용한 해당 기간의 코스피 데이터 가져온 후 엑셀파일로 만들어 다운로드
